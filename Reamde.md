# 🪟 الوكيل للستائر

> موقع ويب احترافي لمحل الوكيل للستائر — بيع، تفصيل، وتركيب الستائر بالقاهرة، مصر.

---

## 📋 نظرة عامة

موقع ويب واحد الصفحة (Single Page Application) مبني بـ HTML / CSS / JavaScript خالص مع دعم PWA وتأثيرات بصرية حديثة.

### المميزات الجديدة
- ✅ واجهة عربية كاملة RTL بتصميم فخم وإضاءات تفاعلية.
- ✅ إضافة خدمات حديثة: الستائر الذكية وغسيل الستائر.
- ✅ نموذج حجز يفتح واتساب مباشرة على الرقم المخصص.
- ✅ دمج رابط مباشر لصفحة الفيسبوك الخاصة بالمحل.
- ✅ PWA — يعمل أوفلاين.
